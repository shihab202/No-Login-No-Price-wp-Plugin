Hides price for logged out/ non users. Product is purchasable once logged in.


== Installation ==


e.g.

1. Upload `No Login No Price.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress


Features:

1. Hides Price and Add to cart button from front end for non logged users
2. Product is unpurchasable for non logged users
3. Price can be seen once user is successfully logged in.
